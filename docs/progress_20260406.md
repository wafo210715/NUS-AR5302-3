### Key Decisions
1. **LDA confirmed** as methodology — best fit for unsupervised, no-ground-truth, interpretable analysis
2. **K range: 4-8** — selected via topic coherence (C_v), following Lin et al. 2025
3. **Exclude residential POIs** — research question focuses on destinations, not residences
4. **Compute and flag mapping bias** — don't adjust. Per-station quality metrics (contributor count, version, POI density)
5. **Frame honestly** — "analysis of OSM-mapped urban environment" not definitive classification
6. **Don't use osmextract** — use Overpass API (overpass-api.de) for POI download
7. **ohsome API for temporal aggregates** — 3 nationwide queries (POI growth, mapper activity, edit type rates); Overpass for element-level data
