# Progress Log: 2025-03-25

## Project Overview

### Title

**Beyond the Campus: Mapping University Community Travel Patterns and Urban Destinations**

### Description

This study examines travel patterns of university communities based on public transport flows near major campuses. By combining origin-destination (OD) data with points-of-interest (POI) data, we analyze where university communities travel and the characteristics of these destinations.

### Research Question

**How do travel patterns differ across university communities, and what characteristics define the destinations they frequent?**

## Methodology Discussion

### The Challenge

"School as Origin" data mixes two populations:

1. **On-campus residents** leaving to go out (to city)
2. **Commuters** leaving to go home (to residential areas)

Data cannot distinguish between them.

### Options Considered


| Option | Approach                                        | Pros                    | Cons                             |
| ------ | ----------------------------------------------- | ----------------------- | -------------------------------- |
| 1      | Filter by destination type (remove residential) | Clean logic             | Mixed-use areas hard to classify |
| 2      | Filter by time-of-day                           | No external data needed | Assumption-based                 |
| 3      | Accept noise, focus on comparison               | Simple, robust          | Can't claim absolute numbers     |


### Decision

**Go with Option 3: Accept the noise, focus on comparative analysis**

- Compare NUS vs NTU using the same methodology
- If both have similar "noise" from commuters, the comparison remains valid
- Acknowledge limitation explicitly in methodology section

### Wording Cautions


| Avoid              | Reason                                      | Use Instead                 |
| ------------------ | ------------------------------------------- | --------------------------- |
| "Students"         | OD data includes staff, visitors, residents | "University community"      |
| "Activate"         | Implies causation                           | "Travel to", "frequent"     |
| "Seek" / "Go for"  | Implies intention not in data               | "Travel to", "destinations" |
| "Play" / "Leisure" | Trip purpose unknown                        | "Travel", "mobility"        |
| "Reveal"           | Strong claim                                | "Suggest", "indicate"       |


---

## Limitation to Acknowledge

> The OD data captures trips originating from or destined to stations near university campuses. We refer to these collectively as "university community" travel, though the data may include non-university users such as residents, workers, and through-travelers in the area.
