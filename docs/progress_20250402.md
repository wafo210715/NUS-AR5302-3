#### Key technical decisions
- **Coordinate systems**: Data stored in EPSG:4326 (WGS84, degrees). Buffer created in EPSG:3414 (Singapore SVY21, meters) for accurate 500m distance, then transformed back to 4326 for output.
- **MRT exit collapsing**: LTA data has multiple exits per station. Collapsed via `group_by(station_name) %>% summarise(geometry = st_centroid(st_union(geometry)))` to get one representative point per station.
- **Bus code zero-padding**: LTA GeoJSON has codes like "4168", OD data uses "04168". `sprintf("%05d", ...)` pads to 5 digits.
- **Path resolution**: `PROJ <- normalizePath("..")` ensures paths work regardless of where each team member clones the repo.

## Methodology Note: Transfer Trips
OD data records each public transport leg separately. A bus→MRT trip appears as two independent OD pairs. We focus on station-level analysis (which stations are popular destinations) rather than journey reconstruction. The NUS-vs-NTU comparative framing remains valid since both campuses have the same limitation. Must acknowledge in Method section.
